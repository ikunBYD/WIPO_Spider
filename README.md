# WIPO 验证码破解模块（spider_gun）

## 模块简介
该模块用于自动化处理 WIPO 网站图片验证码，实现从会话初始化、验证码提取、图片解析、智能识别到模拟提交的全流程自动化，最终返回可直接使用的有效登录会话。

## 核心功能
- 构造合法 Cookie，建立有效会话
- 自动请求页面并提取验证码题目与图片信息
- Base64 编码图片解析与本地保存
- 支持本地模型识别，可兼容 OCR 接口、第三方识别等方案
- 模拟人工点击行为，提交验证码
- 验证通过后返回可用 Session

## 核心执行流程
1. 获取/构造合法 Cookie
2. 请求目标页面，提取验证码信息
3. 解析 Base64 图片并保存至本地
4. 调用识别模型/接口完成验证码识别
5. 构造模拟点击请求，提交验证码
6. 校验验证结果
7. 返回可用 Session

## 实操注意事项
- 验证码通过后，建议重新捕获页面关键参数：`vid`、`ViewState`、Cookie 等
- 识别失败可增加重试机制，提高整体通过率
- 每次验证流程尽量独立会话，避免缓存干扰
